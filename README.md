# Adjustable Heater - Sandboxels Mod

Heats nearby elements up to a set temperature (but not above). Prompts for target temperature when placed.

## Load URLs

**jsDelivr CDN (recommended):**
```
https://cdn.jsdelivr.net/gh/zampxdev/adjustableheater@main/adjustable_heater.js
```

**GitHub Raw:**
```
https://raw.githubusercontent.com/zampxdev/adjustableheater/main/adjustable_heater.js
```

## How to Install

1. Go to https://sandboxels.r74n.com/
2. Click **Mods** button
3. Paste one of the URLs above
4. Press Enter, then refresh the page

## Usage

Find **Adjustable Heater** in the **machines** category. When you place it, a prompt will ask for the target temperature.
